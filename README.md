# Panda TV Clone

팬더티비(Panda TV) 홈 화면 클론 코딩 프로젝트입니다.

## 🚀 실행 방법 (How to Run)

이 프로젝트를 다운로드받아 실행하려면 다음 단계들을 따라주세요.

### 1. 저장소 복제 (Clone)
터미널을 열고 원하는 폴더에서 다음 명령어를 입력해 프로젝트를 다운로드합니다.
```bash
git clone https://github.com/calce2/panda-clone.git
cd panda-clone
```

### 2. 패키지 설치 (Install)
프로젝트 실행에 필요한 라이브러리들을 설치합니다. (Node.js가 설치되어 있어야 합니다)
```bash
npm install
```

### 3. 개발 서버 실행 (Run)
설치가 완료되면 개발 서버를 실행합니다.
```bash
npm run dev
```

### 4. 확인
터미널에 표시된 주소(보통 `http://localhost:5173`)를 브라우저에 입력하여 확인합니다.

## 🛠️ 기술 스택
- React
- Vite
- Vanilla CSS (Dark Theme)
